# MNIST-Dataset-Adversarial-Generation
This repository contains information and material for the MNIST Dataset Adversarial Generation, a personal project. The purpose of the project is to utilize the MNIST data in order to generate fake data identical to the real ones with custom Generative Adversarial Networks. The project was written in Python. Main tasks where data splitting, data scaling, and constructing GAN architectures.

# The dataset can be found in: https://archive.ics.uci.edu/dataset/683/mnist+database+of+handwritten+digits
